Quick Start

Last updated: June 22nd

LineCMS is a Content Management System built using a Flask framework (Python 3.*), LineCMS Provides Custom Themes with Jinja Templating engine, Shop functionality with Stripe's Payment Gateway and a blogging system, We also Use SQLite3

Download

Download the latet version of LineCMS

Download LineCMS

Installation

Download requirements

LineCMS requires some python dependency,

Install requirements using pip (LineCMS Requires Python >= 3.x.x)

Navigate to LineCMS folder, Console:

pip install requirements.txt

Run in Debug mode (Development)

Tweaks and test the CMS

Console

export Debug mode and Run

Navigate to LineCMS folder, Console:

> export FLASK_DEBUG=1

> flask run

Templates

Templates and resources files are split into 2 folders,
static is where your resources (JS/HTML/Images etc..)
static/SYSTEM/custom-theme/
templates folder is where your HTML files should be
templates/SYSTEM/custom-theme/

Folder name should have the same name and a Theme.py where the theme information resides

To create your own theme folder you can copy the whole 'blank-theme' inside the LineCMS, update it and compress into .zip file then move the static/templates to its folders or simply Upload the zip file using dashboard

Linking resources

Linking CSS,JS and other files into your HTML files


Use of static_link(from='custom-theme',get='style.css') template function, more detailed information about this function below.
<!DOCTYPE html> 
<html lang="en">  
<link rel="stylesheet" href="{{ static_link('custom-theme','style.css') }}">

Template functions

Functions below will only work within HTML Files

Admin button

admin_button()
Shows a hyperlink button to Dashboard if the 'logged' session exists (Logged in), returns string (HTML button)

Linking Resources

static_link(from='my-template',get='file.js')
Loads the 'file.js' from 'my-template'

Available parameters:

  • from: (string) required, template name
  • get: (string) a file name you'd like to load (for JS, CSS and other resources files)
  • get-image: (string) an Image file name you'd like to load, 'get' parameter will be ignored.

Load Products

loadproducts(count=12,category="tech")
Shows '12' products under 'tech' category, returns iterable tuple

Available parameters:

  • count: (integer) required, count of products to load
  • category: (string) optional, loads under the specified category

Email templates (Enabled SMTP Required)

You will need to set-up your SMTP Settings enable to use the Templates below which will be sent to the customer's email

SMTP settings can be updated thru Dashboard > Products > Settings and look for the Email Settings

Email templates can be editted from Dashboard > Products > Settings

Fulfilled Template: Is the template used when a Customer order has been Fulfilled (Completed)

  • {{ COMPANYNAME }}: Loads the Name of your website
  • {{ COMPANYNUMBER }}: Loads the Phone Number of your website
  • {{ COMPANYEMAIL }}: Loads the Email of your website
  • {{ TRACKINGLINK }}: Loads the Tracking link you supplied (Will show if not empty)
  • {{ ORDERNUMBER }}: Loads a unique system generated order number
  • {{ ORDERTOTAL }}: Loads the amount of orders (USD) from Stripe
  • {{ ORDERDATE }}: Loads the creation date of the order
  • {{ CUSTOMERNAME }}: Loads the customer name

Placed Template: Is the template used when a Customer payment has been placed

  • {{ COMPANYNAME }}: Loads the Name of your website
  • {{ COMPANYNUMBER }}: Loads the Phone Number of your website
  • {{ COMPANYEMAIL }}: Loads the Email of your website
  • {{ ORDERTOTAL }}: Loads the amount of orders (USD) from Stripe
  • {{ ORDERDATE }}: Loads the creation date of the order
  • {{ CUSTOMERNAME }}: Loads the customer name
